### 与开源版本的改动点
1. 移除了多进程，配置文件中的work_count不再有效
2. 在命令行窗口启动jd-seckill.exe --address 可展示当前账号的所有地址
3. 增加了plus积分的提示，每天早晨自动执行，会输出是否有可加分项，比方说定期的人脸识别。
4. 增加了h5st，**需安装nodejs**，教程网上很多，总之需要在命令行中输入 `node -v` 能显示版本号
